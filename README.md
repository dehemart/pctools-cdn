## üóíÔ∏è **INSTALLATION**

### local installation:

1. clone the repo

```
git clone https://github.com/dehemart/pctools-back
```

2. cd into cloned repo

```
cd pctools-back
```

3. install dependencies

```
yarn
```

4. run the app

```
docker compose up -d
```

```
yarn dev
```
